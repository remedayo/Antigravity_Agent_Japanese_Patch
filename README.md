# Antigravity 日本語化パッチ

このフォルダにある **「Antigravity日本語化ツール.exe」** を使うだけで、直感的に日本語化を適用・解除できます。

## 🚀 手順

1. **Antigravity を完全に終了**します。
2. **`Antigravity日本語化ツール.exe`** をダブルクリックして実行します。
3. ツールが起動したら、**「🇯🇵 日本語にする」** または **「🇺🇸 英語に戻す」** ボタンをクリックするだけです。

---

※ 画面の表示が崩れるなどの異常が起きた場合は、再度ツールを起動して「英語に戻す」を実行してください。
